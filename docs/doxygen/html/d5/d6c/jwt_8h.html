Detailed Description

JWT (JSON Web Token) generation and verification.

Supports HS256 (HMAC-SHA256) signed tokens only. Wrapper over OpenSSL, following the cwist security module convention.

Verify a JWT token and return the decoded claims.

Validates:

  • Structural integrity (three Base64URL-encoded parts).
  • HMAC-SHA256 signature.
  • "exp" claim (token must not be expired) if present.
Parameters
tokenNull-terminated JWT string ("header.payload.sig").
secretHMAC secret key (null-terminated).
Returns
Pointer to a cwist_jwt_claims object on success, NULL on failure. Caller must free with cwist_jwt_claims_destroy().
