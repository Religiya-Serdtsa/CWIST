cwist_heal_result_t cwist_json_heal ( const char * input,
const cwist_heal_config_t * cfg )

Recover malformed JSON using syntax repair, schema alignment, and an optional callback.

Attempt to heal a potentially broken JSON string.

The function first attempts a direct parse, then falls back to the internal L1 syntax healer, and finally consults the user-provided SLLM callback when configured. Each successful step records the recovery level, confidence, and a short operator-facing log of the transformations that were applied.

Parameters
inputRaw JSON text that may contain structural defects.
cfgOptional recovery configuration. NULL applies the built-in defaults.
Returns
Result structure describing the best recovery path that succeeded.
